# 剪绳子问题

# 题目描述
[剑指 Offer 14- I. 剪绳子](https://leetcode.cn/problems/jian-sheng-zi-lcof/)
​

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 1813718137 。请问 $k[0]*k[1]*...*k[m - 1] $可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

**示例 1：**

```bash
输入: 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1
```

**示例 2:**

```bash
输入: 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36
```

- `2 <= n <= 58`

# 动态规划

这道题给定一个大于 1 的正整数 n，要求将 n拆分成至少两个正整数的和，并使这些正整数的乘积最大化，返回最大乘积。
