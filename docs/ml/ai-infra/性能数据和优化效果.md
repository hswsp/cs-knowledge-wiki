## 6.9.4 延迟分析
Mooncake 的延迟特性分析：

TTFT （ Time To First Token ）：

+ 短输入（ <1K ）： < 100ms
+ 中输入（ 1K-8K ）： 100-500ms
+ 长输入（8K-32K）：500ms-2s
+ 超长输入（>32K）：2-10s

TPOT （ Time Per Output Token ）：

+ 基础延迟： 10-50ms/token
+ 受 batch size 影响：batch越大，TPOT越高
+ PD分离后：Decode节点专注于TPOT优化

优化效果：

+ PD分离使 TTFT和 TPOT可以独立优化
+ Prefix Caching显著降低长输入的TTFT
+ 异步传输隐藏传输延迟
