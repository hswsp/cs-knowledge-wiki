# 2.4 素理想与极大理想

为了不把所有理想相关的内容放在同一节，我们在适当的时候分出一节来，以讨论素理想与极大理想的名
义，将其它一些有关理想本身的重要定义、结论放在这一节中讲。类似的处理方法我们在第一章的“有限群”一
节中也见过。为了将纷乱的定义和结论有机地结合起来，对初学者更友好，以一种讲故事的方法智慧地串联知
识，希望大家可以理解这种编排知识的方式。同样地，虽然叫素理想与极大理想，但范围远远不止这两个概念。
我们先定义整环。
注意到在整数环 (Z, +, ·) 中，我们有
∀𝑎, 𝑏 ∈ Z, (𝑎𝑏 = 0 =⇒ 𝑎 = 0 或𝑏 = 0)                                    (2.119)
可是在 (Z6 , +, ·) 中
2·3=0                                              (2.120)

或者严谨地说，
(2 + 6Z)(3 + 6Z) = 0 + 6Z                                     (2.121)

这个重要的区别给出了一个定义，我们称满足上面这条性质的非零交换环为整环。





定义 2.18
令 (𝑅, +, ·) 是一个环，则我们称 𝑅 是个整环，若它是个非零交换环，且没有零因子，即
𝑅 ≠ {0}         (2.122)
𝑅 是个交换环。              (2.123)
∀𝑎, 𝑏 ∈ 𝑅, (𝑎𝑏 = 0 =⇒ 𝑎 = 0 或𝑏 = 0)         (2.124)
若 𝑎 ≠ 0 满足 ∃𝑏 ≠ 0 使得 𝑎𝑏 = 0，我们就称其为一个零因子。
♣

注意到在环 Z6 中，2 和 3 就是零因子。（严格地说是 2 + 6Z 和 3 + 6Z，为了方便我们简称为 2 和 3）。
什么是素理想呢？我们来看整数环上的例子。
根据初等数论，我们知道

引理 2.7
若 𝑝 是一个素数，𝑎, 𝑏 ∈ Z，则
𝑝|𝑎𝑏 ⇐⇒ 𝑝|𝑎 或𝑝|𝑏                     (2.125)
♡

注意到这个命题对于合数 𝑛 是错的，即

引理 2.8
若 𝑛 是一个合数，则存在 𝑎, 𝑏 ∈ Z，使得
𝑛|𝑎𝑏                         (2.126)
𝑛∤𝑎                          (2.127)
𝑛∤𝑏                          (2.128)
♡

证明 证明是简单的。若 𝑛 是一个合数，我们可以取一个非平凡分解 𝑛 = 𝑎𝑏，其中 𝑎, 𝑏 ≠ ±1。则 𝑛|𝑎𝑏，可是
|𝑛| > |𝑎|，故 𝑛 ∤ 𝑎（因为若一个数整除另一个数，则这个数的绝对值必须小于等于另一个数）
。同理 𝑛 ∤ 𝑏。这样，
我们就证明了这个引理。
根据上面这两个引理，我们就知道了，在整数中，这个性质是只对素数成立的。实际上，如果我们用理想来
看，这个性质也可以改为


∀𝑎, 𝑏 ∈ Z, (𝑎𝑏 ∈ 𝑝Z ⇐⇒ 𝑎 ∈ 𝑝Z 或𝑏 ∈ 𝑝Z)             (2.129)

因此，在一般的交换环中，我们定义素理想就是既满足这个性质（如果两个元素的乘积在理想中，则至少有
一个元素在这个理想中），又不是整个环的理想。

定义 2.19
令 (𝑅, +, ·) 是一个交换环，而 𝔭 ◁ 𝑅，则我们称 𝔭 是个素理想，若
∀𝑎, 𝑏 ∈ Z, (𝑎𝑏 ∈ 𝔭 ⇐⇒ 𝑎 ∈ 𝔭 或𝑏 ∈ 𝔭)         (2.130)
𝔭≠𝑅           (2.131)
♣

对代数结构敏感的同学可能已经注意到了素理想和整环之间的微妙联系。事实上，我们通过一个命题来将
谜底揭开。

命题 2.20
令 (𝑅, +, ·) 是一个交换环，而 𝔭 ◁ 𝑅。则 𝔭 是一个素理想，当且仅当商环 𝑅/𝔭 是一个整环。
♠





证明 先证充分性。令 𝔭 是一个素理想。因为 𝑅 是交换环，则显然 𝑅/𝔭 也是交换环。因为对 𝑎, 𝑏 ∈ 𝑅，我们有
(𝑎 + 𝔭)(𝑏 + 𝔭) = 𝑎𝑏 + 𝔭 = 𝑏𝑎 + 𝔭 = (𝑏 + 𝔭)(𝑎 + 𝔭)            (2.132)
而且因为 𝔭 ≠ 𝑅，所以 𝑅/𝔭 不是零环。
我们只须证明 𝑅/𝔭 中没有零因子。假设
(𝑎 + 𝔭)(𝑏 + 𝔭) = 0 + 𝔭                          (2.133)
则 𝑎𝑏 ∈ 𝔭。根据 𝔭 是素理想，不失一般性假设 𝑎 ∈ 𝔭。则
𝑎 +𝔭 = 0+𝔭                                 (2.134)
这就证明了 𝑅/𝔭 是一个整环。
再证必要性。假设 𝑅/𝔭 是一个整环。类似地，我们知道因为 𝑅/𝔭 不是零环，所以 𝔭 ≠ 𝑅。
再令 𝑎, 𝑏 ∈ 𝑅，使得 𝑎𝑏 ∈ 𝔭，则
𝑎𝑏 + 𝔭 = (𝑎 + 𝔭)(𝑏 + 𝔭) = 0 + 𝔭                     (2.135)
由于 𝑅/𝔭 是一个整环，故不失一般性假设 𝑎 + 𝔭 = 0 + 𝔭，这就证明了 𝑎 ∈ 𝔭，即 𝔭 是一个素理想。
讲了素理想，我们来讲极大理想。

定义 2.20
令 (𝑅, +, ·) 是一个交换环，而 𝔪 ◁ 𝑅。则我们称 𝔪 是一个极大理想，若 𝔪 ≠ 𝑅，且它是个极大的理想，即
对于任意 𝐼 ◁ 𝑅，如果 𝐼 ⊋ 𝔪，则
𝐼=𝑅                                  (2.136)

这就是说，唯一严格大于 𝔪 的理想，是整个环。
♣

实际上，正如素理想的充要条件，我们也有极大理想的充要条件。

命题 2.21
令 (𝑅, +, ·) 是一个交换环，而 𝔪 ◁ 𝑅。则 𝔪 是一个极大理想，当且仅当商环 𝑅/𝔪 是一个域。
♠

证明 先证充分性。令 𝔪 是一个极大理想。因为 𝑅/𝔪 是交换环，我们只须证明每个非零元素都有逆元。令
𝑎 + 𝔪 ∈ 𝑅/𝔪(𝑎 + 𝔪 ≠ 0 + 𝔪)，也就是说 𝑎 ∉ 𝔪。只须证明存在 𝑏 + 𝔪 ∈ 𝑅/𝔪(𝑏 ∈ 𝑅)，使得 𝑎𝑏 + 𝔪 = 1 + 𝔪。
等价地，我们只须证明存在 𝑏 ∈ 𝑅, 𝑚 ∈ 𝔪，使得
1 = 𝑎𝑏 + 𝑚                                 (2.137)
考虑到 𝑎 ∉ 𝔪，所以 𝔪 + 𝑅𝑎 = (𝔪, 𝑎)，是一个严格包含了 𝔪 的理想。因为 𝔪 是极大理想，所以 𝔪 + 𝑅𝑎 = 𝑅。
右边取 1 ∈ 𝑅，我们就得到了，存在 𝑏 ∈ 𝑅，使得 1 = 𝑎𝑏 + 𝑚，这就证明了充分性。
再证必要性。如果 𝑅/𝔪 是一个域，那么对于假设理想 𝐼 ⊋ 𝔪。任取 𝑎 ∈ 𝐼 \ 𝔪。则 𝑎 + 𝔪 ≠ 0 + 𝔪，故存在
𝑏 ∈ 𝑅，使得 𝑎𝑏 + 𝔪 = 1 + 𝔪。因此，也存在 𝑚 ∈ 𝔪，使 1 = 𝑎𝑏 + 𝑚。
因此，对任意 𝑟 ∈ 𝑅，我们都有
𝑟 = 𝑟 (𝑎𝑏 + 𝑚) = 𝑟𝑎𝑏 + 𝑟𝑚 ∈ 𝐼𝑏 + 𝔪 ⊂ 𝐼 + 𝐼 = 𝐼              (2.138)
这就证明了 𝐼 = 𝑅。因此 𝔪 是一个极大理想。
综上所述，我们就证明了这个命题。
我们想要证明每一个极大理想都是素理想。一个便捷的方法是证明每一个域都是整环。

引理 2.9
令 (𝑅, +, ·) 是一个域，则 𝑅 是一个整环。
♡

证明 一个域当然是一个交换环。令 𝑎, 𝑏 ∈ 𝑅，使 𝑎𝑏 = 0。我们只须证明 𝑎 = 0 或 𝑏 = 0。

1 = 11 = 𝑎𝑐𝑏𝑑 = 𝑎𝑏𝑐𝑑 = 0𝑐𝑑 = 0               (2.139)
于是 𝑅 = {0}。这和我们假设的存在 𝑎 ≠ 0 是矛盾的。因此每一个域都是整环。
利用这个引理，我们就得到下面的推论。
