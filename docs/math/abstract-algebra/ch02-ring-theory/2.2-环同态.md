# 2.2 环同态

因为第一章的铺垫足够多，所以我们可以快速地进入环同态的讨论。更妙的是，环同构第一定理与群同构第一定理非常相似（在加法上是一致的，而在乘法上还是有微妙的区别的）

。这意味着我们讨论在环同态上不会花太多的时间。

**定义 2.9**

令$f : (R, +, \cdot) \to (R', +′, ∗)$是一个映射，我们说f是个环同态，若f (1) = 1′             (2.29)

f (a + b) = f (a) +′ f (b)                   (2.30)

f (ab) = f (a) ∗ f (b)                  (2.31)

未来，在不引起歧义的情况下，我们会忽略两个环中加法与乘法的区别，都记作+和$\cdot$，称环同态是$f : (R, +, \cdot) \to (R', +, \cdot)                     (2.32)$

注意到，这其实就是说，f 既是加法的群同态，又是乘法的幺半群同态。不要忘记，幺半群同态我们是要保持单位元的。而群同态因为逆元的存在，不需要额外的保持单位元的条件。这是个很小的细节。但通过这个细节，就能看出你是否真正理解了幺半群和群的重要区别。

要注意，因为我们定义了环是有乘法单位元的，因此 ker( f ) 不一定是子环。所以，假如我们用群论作对比，在群论中正规子群是子群，可是在环论中的“正规子群”式的结构甚至几乎永远不是子环（除非是整个环）

。我们管环论中的“正规子群”式的结构叫“理想”。这也是一个非常有名的概念，当然很多人只是听过了这个术语而已。和群论类似，im( f )依然是子结构（在这里是子环）。那么，我们希望有的环同构第一定理，便是希望$R/ker( f ) \sim eq im( f )                        (2.33)$

理想在一定程度上，是一个“奇怪的”概念，原因便是初学者是很难一下子理解这样定义的好处。当时讲正规子群的时候，我们可以说因为希望左陪集的乘法是良定义的。这里虽然也是一样的道理，但是定义方法却并不类似。我们要提醒大家的是，理想这个概念，和数学中许多重要的概念一样，是经历了几年乃至几十年的研究后，才慢慢确定下来的——因此初学者不必立刻“理解”。事实上，在学完环同构第一定理的瞬间，应当就对这个概念有一定认识了。在学习得越来越深入后，对这一概念的熟悉便加深了。实际上，对所有抽象概念，人的认知都有类似的过程。我们铺垫这么多，就是为了在很大程度上提醒大家，下一个定义的重要性。

**定义 2.10**

令$(R, +, \cdot)$是一个环，而$I \subset R$。我们定义，称I是R的左理想，若(I, +) < (R, +)                        (2.34)

$$
\forall r \in R, \forall a \in I, ra \in I                         (2.35)
$$

类似地，我们称 I 是 R 的右理想，若(I, +) < (R, +)                        (2.36)

$$
\forall a \in I, \forall r \in R, ar \in I                         (2.37)
$$

如果 I 既是左理想又是右理想，我们就称 I 是 R 的一个理想，记作 I ◁ R。

我们刚才提过，一个理想一般不是子环。实际上，一个理想是子环当且仅当它是整个环。

**引理 2.2**

令$(R, +, \cdot)$是一个环，而I ◁ R。则I < R当且仅当I = R。

**证明** 必要性是显然的，因为一个环当然是自己的子环。

我们来证明充分性。假设I < R，则特别地，$1 \in I$。可是I ◁ R，因此对任何$r \in R$，我们有$r =r \cdot 1\in I                                   (2.38)$

这就证明了 I = R。

综上所述，一个理想是子环当且仅当它是整个环。

注意，对于一个交换环来说，左理想等价于右理想，也就等价于理想。我们只需要证明左理想或右理想中的一条就可以证明另一条。

**命题 2.6**

假设$(R, +, \cdot)$是一个交换环，则I是一个左理想当且仅当I是一个右理想，又当且仅当I是一个理想。

证明根据交换环对乘法的交换律，这是显然的。

我们可以说，理想的第二条性质指的是：理想在乘法下“吸收”了整个环到理想上，也就是说$RI \subset I                                        (2.39)$

$$
IR \subset I                                        (2.40)
$$

其中子集的乘法，当然指所有元素乘积的集合。

我们举一个重要的例子：nZ 是 Z 的理想。通过这个例子，我们来体会一下，理想是怎么“吸收”整个环的乘法到其自身的。

**命题 2.7**

令$n \in N1$，则nZ是Z的理想，即nZ ◁ Z                                     (2.41)

**证明**首先，我们知道(nZ, +)是(Z, +)的（加法）子群。这可以通过构造$m \mapsto\to mn$的（加法）群同态来得到（我

们上一章证明过）。

其次，注意到Z是一个交换环，故我们只须证明$RI \subset I$，在这里就是$Z \cdot nZ \subset nZ$。什么意思呢？就是说，一个整数乘上一个n的倍数，还是n的倍数。这是显然的，但这种显然蕴藏着一种微妙。数学家们通过体会这一种微妙，找到了理想的正确定义。这告诫我们，不要被看似简单的事实或现象蒙蔽双眼。我们应当从中找到数学的规律，而不是找到自己的傲慢。

要证明$Z \cdot nZ \subset nZ$，我们只须令$m \in Z$，$nk \in nZ(k \in Z)$，只要证明$mnk \in nZ$即可。而这是因为$mnk = n(mk) \in nZ                                     (2.42)$

综上所述，这就证明了 nZ 是 Z 的理想。

我们发现，正如正规子群可以从群同态中定义（群同态的核），理想也可以从环同态中定义（环同态的核）。

我们用一个小引理来强调这个事实。

**引理 2.3**

令$n \in N1$，我们要定义映射$f : (Z, +, \cdot) \to (Zn , +, \cdot)$。对$m \in Z$，我们定义f (m) = m + nZ                                 (2.43)

则 f 是一个环同态，而 ker( f ) = nZ ◁ R。

证明我们在上一章中证明了 f 是加法的群同态。因此只须证明 f 是乘法的幺半群同态。

第一， f (1) = 1 + nZ 是 Zn 的乘法单位元。

第二，若$m, m' \in Z$，则利用上一章中我们证明过的Zn对乘法的良定义性，我们有f (m) f (m') = (m + nZ) (m' + nZ) = mm' + nZ = f (mm')             (2.44)

综上所述， f 是一个从 Z 到 Zn 的环同态。

事实上，通过这个例子，我们想要引出的便是，每一个环同态的核都是理想。刚才我们也说了，每一个环同态的像都是子环。现在，我们来证明。

**命题 2.8**

令$f : (R, +, \cdot) \to (R', +, \cdot)$是一个环同态，则f的核是R的理想，f的像是R'的子环。此即，$ker( f ) = {a \in R : f (a) = 0′ } ◁ R     (2.45)$

$$
′                                  '              'im( f ) = {b \in R : \exists a \in R, b = f (a)} = { f (a) \in R : a \in R} < R        (2.46)
$$

**证明** 我们先证明 ker( f ) ◁ R。根据群同态的性质，我们知道 ker( f ) 是加法的（正规）子群。为了方便起见，令

I = ker( f )。我们只须证明$RI \subset I$以及$I R \subset I$。

令$a \in R$，$b \in I = ker( f )$，故f (b) = 0′。因此，f (ab) = f (a) f (b) = f (a)0′ = 0′，从而$ab \in ker( f ) = I$。这就证明了$RI \subset I$。而另一个包含关系是非常类似的，这里略去证明。这样，我们就证明了ker( f ) ◁ R。

我们再证明im( f ) < R'。第一，$1′ = f (1) \in im( f )$。

第二，令$a', b' \in im( f )$，不妨设a' = f (a)，b' = f (b)。只须证明$a' − b', a' b' \in im( f )$。而这分别是因为$a' − b' = f (a) − f (b) = f (a − b) \in im( f )                                   (2.47)$

$$
a' b' = f (a) f (b) = f (ab) \in im( f )                                (2.48)
$$

这就证明了 im( f ) < R'。

综上所述，我们证明了这个命题。

接下来，我们想要给出商环的定义。注意，对加法群而言，商群是由加法的陪集所构成。在环中，我们既有加法，又有乘法，但是商环（群）仍然是由加法的陪集所构成。这是非常重要的一个出发点：商环，从集合意义上来说，等于加法的商群。从这个出发点开始，我们来定义环对于理想的商环。

**定义 2.11**

令$(R, +, \cdot)$是一个环，而I ◁ R。我们定义R对I的商环，定义为$(R/I, +, \cdot)$，其中$R/I = {a + I : a \in R}                                             (2.49)$

而加法和乘法分别对$a + I, b + I \in R/I (a, b \in R)$，定义为(a + I) + (b + I) = (a + b) + I                                        (2.50)

(a + I) (b + I) = (ab) + I                                       (2.51)

这个定义是不良好的。我们将其良定义性，和商环是个环这个事实，一并写进下一个命题中。

**命题 2.9**

令$(R, +, \cdot)$是一个环，而I ◁ R，则上述的加法和乘法是良定义的，且商环$(R/I, +, \cdot)$是一个环。

**证明** 因为环对加法构成阿贝尔群，而理想对加法构成子群，因此理想对加法构成正规子群。根据上一章的内容，

正规子群这样的陪集加法是良定义的。

我们要证明商环对乘法是良定义的。令a + I = a' + I，b + I = b' + I，即$a − a' \in I, b − b' \in I$。我们只须证明ab + I = a' b' + I，即$ab − a' b' \in I$。而这是因为$ab − a' b' = (ab − a' b) + (a' b − a' b') = (a − a')b + a' (b − b') \in I R + RI = I + I = I       (2.52)$

其中倒数第二个等号是根据理想对乘法的“吸引”性质，而最后一个等号是根据理想对加法的封闭性。这样，我们就证明了商环对乘法是良定义的。

接下来，要证明商环是个环，其实只要将 R 上环的结构（利用良定义性）照搬过来即可。

利用 I 对加法构成正规子群，因此利用上一章的内容，R/I 对加法构成群。我们只须证明 R/I 对乘法构成幺半群，且乘法对加法有左右分配律。

乘法单位元是1 + I，因为对任意$a + I (a \in R)$，我们有(a + I)(1 + I) = (1 + I)(a + I) = a + I                                        (2.53)

R/I对乘法有结合律，这是因为对任意$a + I, b + I, c + I (a, b, c \in R)$，我们有((a + I) (b + I)) (c + I) = (ab + I) (c + I) = (ab)c + I = a(bc) + I = (a + I) ((b + I)(c + I))             (2.54)

最后，我们要证明乘法对加法有左右分配律。利用对称性，我们只证明左分配律。对任意$a+I, b+I, c+I (a, b, c \inR)$，我们有(a + I)((b + I) + (c + I)) = (a + I)((b + c) + I) = a(b + c) + I = (ab + ac) + I = (a + I) (b + I) + (a + I) (c + I) (2.55)

综上所述，我们就证明了 R/I 是个环。这个环被叫做 R 对 I 的商环。

**定义 2.12**

令$f : (R, +, \cdot) \to (R', +, \cdot)$是一个映射，我们称f是一个环同构，若f既是双射，又是环同态。

和之前一样，我们应当出于对对称性的考虑，证明每个环同构 f 的逆映射仍然是环同态，进而也是环同构。

**引理 2.4**

令$f : (R, +, \cdot) \to (R', +, \cdot)$是个环同构，则f −1是个环同态，进而也是环同构。

**证明** 事实上， f 是环同构，当且仅当 f 对加法是群同构，而对乘法是幺半群同构。因为我们对群同构和幺半群

同构都证明了逆映射仍然是同构。因此环同构的逆映射也是环同构，也就是环同态。这就证明了这个引理。

现在，我们终于做好环同构第一定理的准备了。让我们毫不犹豫地开始证明环同构第一定理吧！

**定理 2.1** (环同构第一定理)

令$f : (R, +, \cdot) \to (R', +, \cdot)$是一个环同态，则R对ker( f )构成的商环，同构于im( f )。此即，$R/ker( f ) \sim eq im( f )                                                (2.56)$

证明我们的定义方式和群同构定理是一致的。令$f˜ : R/ker( f ) \to im( f )$，对a + ker( f )，定义为f˜(a + ker( f )) = f (a)                                                (2.57)

我们在上一章证明了 f˜ 是良定义的，且对加法构成群同构。要证明 f˜ 是环同构，只须证明它对乘法是幺半群同态。

令$a + ker( f ), b + ker( f ) \in R/ker( f )(a, b \in R)$，则f˜((a + ker( f )) (b + ker( f ))) = f˜(ab + ker( f )) = f (ab) = f (a) f (b) = f˜(a + ker( f )) f˜(b + ker( f ))       (2.58)

综上所述， f˜ 给出了一个从商环 R/ker( f ) 到像 im( f ) 的环同构。这就证明了这个定理。

我们发现，上面的证明其实是之前的众多命题所铺垫的。真正证明的时候，已经非常简单了。这也是数学学习或研究的一个方法。我们只要把基础打扎实了，证明定理的时候，就不会很困难。

下面是环同构第二与第三定理。

**定理 2.2** (环同构第二定理)

令$(R, +, \cdot)$是一个环，而S < R，I ◁ R。则S + I < R，$S \cap I ◁ S$，I ◁ S + I，且$S/(S \cap I) \sim eq (S + I)/I                                                 (2.59)$

**证明** 我们先证明 S + I < R。对加法而言，S 和 I 都是子群，因此我们只须证明 S + I 对乘法构成子幺半群，即对

乘法是封闭的，且包含单位元。第一，$1 = 1 + 0 \in S + I$。第二，只须证明$(S + I)(S + I) \subset (S + I)$，而这是因为$(S + I)(S + I) = SS + SI + IS + I I \subset S + I + I + I = S + I       (2.60)$

倒数第二个等号是根据 S 与 I 对乘法的封闭性，及 I 对乘法的“吸收”性。最后一个等号是根据 I 对加法的封闭性。

我们再证明$S \cap I ◁ S$。同样根据上一章，$S \cap I$对加法构成子群。我们只须证明$S \cap I$对乘法的“吸收”性，即$(S \cap I)S \subset S \cap I$，及$S(S \cap I) \subset S \cap I$。根据对称性，我们证明前面这个包含关系。

$$
(S \cap I)S \subset SS \cap IS = S \cap I                         (2.61)
$$

根据对称性，$S \cap I ◁ S$。

我们接着证明I ◁ S + I。根据上一章，加法的子群关系是显然的。我们只须证明$I (S + I) \subset I$，及$(S + I)I \subset I$。

根据对称性，我们证明前面这个包含关系。

$$
I (S + I) \subset IS + I I = I + I = I                    (2.62)
$$

根据对称性，I ◁ S + I。

我们最后证明$S/(S \cap I) \sim eq (S + I)/I$。和群同构第二定理的证明一样，我们定义$f : S \to (S + I)/I$，对$a \in S$，定义为$f (a) = a + I \in (S + I)/I                        (2.63)$

根据上一章的结论，f一个是良定义的满射，对加法构成群同态，且$ker( f ) = S \cap I$。因此我们只要证明f对乘法是幺半群同态，就可以利用环同构第一定理证明这个命题了。而这是显然的，因为若$a, b \in S$，则f (a) f (b) = (a + I)(b + I) = ab + I = f (ab)              (2.64)

因此，由环同构第一定理，我们得到了$S/(S \cap I) \sim eq (S + I)/I                           (2.65)$

综上所述，我们就证明了这个命题。

**定理 2.3** (环同构第三定理)

令$(R, +, \cdot)$是一个环，而I, J ◁ R，且$I \subset J$。则J/I ◁ R/I，且$(R/I)/(J/I) \sim eq R/J                           (2.66)$

证明我们先证明J/I ◁ R/I。对加法而言J是R的子群，因此J/I是R/I的加法子群。我们只须证明$(J/I)(R/I) \subsetJ/I$，及$(R/I)(J/I) \subset J/I$。根据对称性，我们证明前面这个包含关系。因为J ◁ R，所以$(J/I)(R/I) = (J R)/I \subset J/I                         (2.67)$

这就证明了 J/I ◁ R/I。

和上一章一样，我们令$f : R/I \to R/J$，对$a + I (a \in R)$，定义为f (a + I) = a + J                            (2.68)

根据上一章的结论，f是一个良定义的满射，对加法构成群同态，且ker( f ) = J/I。因此我们只要证明f对乘法是幺半群同态，就可以利用环同构第一定理证明这个命题了。而这是显然的，因为若$a + I, b + I \in R/I (a, b \in R)$，则f (a + I) f (b + I) = (a + J)(b + J) = ab + J = f (ab + I)      (2.69)

因此，由环同构第一定理，我们得到了$(R/I)/(J/I) \sim eq R/J                          (2.70)$

综上所述，我们就证明了这个命题。

我们做个简单的复盘。实际上，我们会发现，环同构三定理，在良定义性和加法上，是完全照搬了群同构三定理的结论的，而对乘法的幺半群同态则是几乎显然的。因此，环同构定理实际上是群同构三定理的升级版。

未来我们还会看到模论中的模同构三定理。到时候我们会注意到，证明也是类似的。
