# 第 2 章 环论 I——Ring Theory I

第 2 章 环论 I——Ring Theory I

2.1 环
注意到，环实际上有两种定义。一种是有乘法单位元的，而另一种是没有乘法单位元的。我们在这里假设环
都是有乘法单位元的。

定义 2.1
我们说 (𝑅, +, ·) 是一个环，当 (𝑅, +) 是个阿贝尔群，(𝑅, ·) 是个幺半群，且乘法对加法有左右分配律，即
∀𝑎, 𝑏, 𝑐 ∈ 𝑅, 𝑎(𝑏 + 𝑐) = 𝑎𝑏 + 𝑎𝑐   (2.1)
∀𝑎, 𝑏, 𝑐 ∈ 𝑅, (𝑎 + 𝑏)𝑐 = 𝑎𝑐 + 𝑏𝑐   (2.2)
♣

𝑅 指代的是英文单词 𝑟𝑖𝑛𝑔，即环。
那么什么是一个环呢？我们可以这样理解。对加法封闭，有交换、结合律，有单位元和逆元；对乘法封闭，
有结合律，有单位元（有的定义中不需要有乘法单位元）。乘法不需要有逆元。
同时，我们不要求乘法有交换律。对于那些有乘法交换律的环，我们称之为交换环。

定义 2.2
令 (𝑅, +, ·) 是一个环，我们说 𝑅 是一个交换环，当 𝑅 对乘法有交换律，即
∀𝑎, 𝑏 ∈ 𝑅, 𝑎𝑏 = 𝑏𝑎          (2.3)
♣

注意，在交换环中，因为乘法是交换的，因此，左右分配律变成分配律。
（只要证明任何一边，就可以利用
乘法交换律得到另一边）。
环仍然是很常见的概念。最常见的环是整数环 (Z, +, ·)。不难检验这是一个环。我们结合上一章提到过的。我
们实际上，也知道 (Z𝑛 , +, ·) 是一个环。唯一要检验的便是分配律。考虑到 Z𝑛 中加法和乘法的良定义性，我们可
以直接把 Z 中乘法对加法的分配律直接转移到 Z𝑛 中。具体的证明可以留给大家作为练习。
另一个经典的例子是 𝑛 ∗ 𝑛 实矩阵环 (𝑀 (𝑛, R), +, ·)。这是一个非交换环——乘法是不交换的。实际上，环是
很常见的。仅仅是环的性质其实往往不够好，所以我们会给环加上各种更多的公理，比如得到整环，唯一分解
整环，主理想整环，欧几里得整环，域等等。未来，我们会分别研究这些特别的环，以及它们之间的联系。
正如在幺半群、群论的开始我们证明了一些性质。环论的定义给了以后，有一些简单的小练习，我们用命题
的形式给出。

命题 2.1
令 (𝑅, +, ·) 是一个环，而 𝑎, 𝑏, 𝑐 ∈ 𝑅，则
𝑎0 = 0𝑎 = 0        (2.4)
𝑎(−𝑏) = (−𝑎)𝑏 = −(𝑎𝑏)         (2.5)
(−𝑎) (−𝑏) = 𝑎𝑏        (2.6)
♠

证明 一：首先，利用分配律，
𝑎0 = 𝑎(0 + 0) = 𝑎0 + 𝑎0          (2.7)

因此 𝑎0 = 0。根据对称性，0𝑎 = 𝑎。
二：根据对称性，我们只须证明 𝑎(−𝑏) = −(𝑎𝑏)。而这是因为
𝑎(−𝑏) + 𝑎𝑏 = 𝑎(−𝑏 + 𝑏) = 𝑎0 = 0        (2.8)

三：利用两次性质二，我们就得到
(−𝑎)(−𝑏) = −(𝑎(−𝑏)) = −(−(𝑎𝑏)) = 𝑎𝑏       (2.9)
有一个重要的环是零环，它是最平凡的环，即 (0, +, ·)。它只有一个元素，既是加法单位元也是乘法单位元，
定义为
00 = 0                   (2.10)
0+0=0                       (2.11)
很容易检验这是一个环。
我们要说明的一个结论是，一个环是零环当且仅当加法单位元等于乘法单位元，即 0 = 1。

命题 2.2
令 (𝑅, +, ·) 是一个环，则 𝑅 = {0} 当且仅当 0 = 1。
♠

证明 充分性是显然的。
我们来证明必要性。假设 0 = 1，我们只须证明对所有 𝑎 ∈ 𝑅，都有 𝑎 = 0。
𝑎 = 𝑎1 = 𝑎0 = 0                (2.12)
这就证明了这个命题。
我们在前一章学习了，每一个幺半群的所有可逆元素构成了一个群。由于每一个环 (𝑅, +, ·) 都包含了一个幺
半群 (𝑅, ·)。因此我们也定义所有乘法可逆元素构成的群，记作 𝑅 × 。

定义 2.3
令 (𝑅, +, ·) 是一个环，则 (𝑅 × , ·)，是由 𝑅 中所有乘法可逆元素构成的群。𝑅 中的乘法可逆元素又被称为 𝑅
中的单位。
♣

，0 ≠ 1。因此 0 永远是没有乘法逆元的，这是因为对所有 𝑎 ∈ 𝑅，
我们知道，对于一般的环（不是零环的环）
我们有 0𝑎 = 0 ≠ 1。所以最好的情况可能就是 𝑅 \ {0} = 𝑅 × ，即所有非零元素都有乘法逆元。这样的环的性质是
很好的，我们称它为一个除环。

定义 2.4
令 (𝑅, +, ·) 是一个环，我们称 (𝑅, +, ·) 是一个除环，若
𝑅 \ {0} = 𝑅 ×              (2.13)
也即，所有非零元素都是单位。
♣

不难发现，除环的充要条件是对加法构成阿贝尔群，对乘法构成群，且乘法对加法有左右分配律。
更好的环便是域。一个域是指交换的除环。

定义 2.5
令 (𝑅, +, ·) 是一个环，我们称 (𝑅, +, ·) 是一个域，若它是一个交换的除环。
♣

命题 2.3
(𝑅, +, ·) 是一个域，当且仅当
(𝑅, +) 是一个阿贝尔群               (2.14)
(𝑅 \ {0}, +) 是一个阿贝尔群              (2.15)
乘法对加法有分配律                  (2.16)
♠

证明 根据定义，这是显然的。


我们发现 (𝑍, +, ·) 是交换环，但不是域，因为 2 的乘法逆元 12 显然不是整数。当我们扩充到所有的有理数以
后，(Q, +, ·) 就成为域。类似地，(R, +, ·)，(C, +, ·) 也是域。我们下一个章节就会专门讨论域。域的性质比环好太
多，但这也大大限制了域的范围——域完全没有环常见。因为域是特殊的环，所以环的大多性质都可以转移到
域上，但反过来确实完全不可以的。从另一个方面说，因为域的性质太好，所以可以研究的内容，也和环并不一
样。未来，大家会从学习中体会到这一段中所说的内容。
讲了环，当然要讲子环。

定义 2.6
令 (𝑅, +, ·) 是一个环，而 𝑆 ⊂ 𝑅。我们说 𝑆 是 𝑅 的子环，记作 𝑆 < 𝑅，若
0, 1 ∈ 𝑆                                                    (2.17)
∀𝑎, 𝑏 ∈ 𝑆, 𝑎 + 𝑏, 𝑎𝑏 ∈ 𝑆                                                    (2.18)
∀𝑎 ∈ 𝑆, −𝑎 ∈ 𝑆                                                      (2.19)
♣

事实上，这就是说 (𝑆, +) 是 (𝑅, +) 的子群，(𝑆, ·) 是 (𝑅, ·) 的子幺半群。我们当然有简单一些的等价条件。

引理 2.1
令 (𝑅, +, ·) 是一个环，而 𝑆 ⊂ 𝑅，则 𝑆 < 𝑅 当且仅当
1∈𝑆                                                   (2.20)
∀𝑎, 𝑏 ∈ 𝑆, 𝑎 − 𝑏, 𝑎𝑏 ∈ 𝑆                                                     (2.21)
♡

证明 假如满足了这两个条件，那么 0 = 1 − 1 ∈ 𝑆。而 −𝑎 = 0 − 𝑎 ∈ 𝑆，𝑎 + 𝑏 = 𝑎 − (−𝑏) ∈ 𝑆。这就证明了这是个
子环。
另一个方向是显然的。假如 𝑆 是子环，那么 𝑎 − 𝑏 = 𝑎 + (−𝑏) ∈ 𝑆。
例如 Z 就是 Q 的子环。
有了子环，我们就想知道是否用子集可以生成子环。答案还是肯定的。因为我们已经在幺半群和群中都做
过类似的操作了，所以我们不赘述了。

定义 2.7
令 (𝑅, +, ·) 是一个环，而 𝐴 ⊂ 𝑅，则 𝐴 生成的子环，记作 ⟨𝐴⟩，定义为所有包含了 𝐴 的子环的交集，即
Ù
⟨𝐴⟩ =   {𝑆 ⊂ 𝑅 : 𝑆 ⊃ 𝐴, 𝑆 < 𝑅}  (2.22)
♣

命题 2.4
令 (𝑅, +, ·) 是一个环，而 𝐴 ⊂ 𝑅，则 ⟨𝐴⟩ < 𝑅。
♠

证明 首先这个集族是非空的，因为 𝑅 本身就是一个包含了 𝐴 的子环。
接下来，我们利用上面的引理。令 𝑆 是一个包含了 𝐴 的子环。因为 1 在每一个这样的 𝑆 中，所以 1 ∈ ⟨𝐴⟩。
令 𝑎, 𝑛 ∈ ⟨𝐴⟩，则 𝑎 − 𝑏, 𝑎𝑏 在每一个这样的 𝑆 中，因为每一个 𝑆 都是子环。因此 𝑎 − 𝑏, 𝑎𝑏 ∈ ⟨𝐴⟩。
综上所述，⟨𝐴⟩ < 𝑅。
