# 3.3 不可约多项式

在这一节中，我们会介绍著名的艾森斯坦判别法。这个判别法给出了在唯一分解整环上不可约多项式的充
分条件，而这个充分条件是很好检验的。而且更妙的是，就算多项式本身不能通过这个充分条件的检查，我们
仍然可以对多项式作一些简单的变换，使变换后的多项式满足充分条件，而根据不可约在这类变换下的不变形，
我们就能证明本来的多项式也是不可约的。
下面，我们直接给出这个命题。

命题 3.13 (艾森斯坦判别法)
令 (𝑅, +, ·) 是一个唯一分解整环，而 𝐹 = Frac(𝑅) 是它的分式域。令 𝑓 (𝑥) ∈ 𝑅[𝑥] 是一个非常数多项式，即
deg( 𝑓 ) ⩾ 1。令 𝑛 = deg( 𝑓 )，则 𝑓 (𝑥) 可以写成
𝑓 (𝑥) = 𝑎 𝑛 𝑥 𝑛 + 𝑎 𝑛−1 𝑥 𝑛−1 + · · · + 𝑎 1 𝑥 + 𝑎 0                       (3.143)
假如存在某个素元素 𝑝 ∈ 𝑅，使得
𝑝 ∤ 𝑎𝑛                                     (3.144)
𝑝|𝑎 𝑛−1 , · · · , 𝑝|𝑎 0                                   (3.145)
𝑝 ∤ 𝑎0
(3.146)






则 𝑓 (𝑥) 在 𝐹 [𝑥] 上是不可约的。进一步地，如果 𝑓 (𝑥) 是 𝑅[𝑥] 上的本原多项式，那么 𝑓 (𝑥) 在 𝑅[𝑥] 上也是
不可约的。
♠

证明 用反证法，假设 𝑓 (𝑥) 在 𝐹 [𝑥] 上是可约的，则存在 𝑔(𝑥), ℎ(𝑥) ∈ 𝐹 [𝑥]，使得
𝑔(𝑥) = 𝑏 0 + · · · + 𝑏 𝑚 𝑥 𝑚                              (3.147)
ℎ(𝑥) = 𝑐 0 + · · · + 𝑐 𝑙 𝑥 𝑙                           (3.148)
𝑓 (𝑥) = 𝑔(𝑥)ℎ(𝑥)                                (3.149)
因为 𝑝|𝑎 0 而 𝑝 2 ∤ 𝑎 0 ，所以 𝑝 整除 𝑏 0 和 𝑐 0 中的恰好一个，不失一般性，假设 𝑝|𝑏 0 ，而 𝑝 ∤ 𝑐 0 。接下来，我
们用数学归纳法，证明对于任意 0 ⩽ 𝑘 ⩽ 𝑛 − 1，我们有 𝑝|𝑏 𝑘 。
假设对于所有 0 ⩽ 𝑘 < 𝑙 ⩽ 𝑛 − 1，我们有 𝑝|𝑏 𝑘 ，则当 𝑘 = 𝑙 时，由于
𝑝|𝑎 𝑙 = 𝑏 0 𝑐 𝑙 + 𝑏 1 𝑐 𝑙−1 + · · · + 𝑏 𝑙−1 𝑐 1 + 𝑏 𝑙 𝑐 0             (3.150)
其中 𝑝|𝑏 0 , · · · , 𝑏 𝑙−1 ，所以 𝑝|(𝑏 𝑙 𝑐 0 )。可是 𝑝 ∤ 𝑐 0 ，因此我们有 𝑝|𝑏 𝑙 。这就证明了 𝑝|𝑏 0 , · · · , 𝑝|𝑏 𝑛−1 。
同理，因为 𝑝 ∤ 𝑎 𝑛 ，而
𝑎 𝑛 = 𝑏 0 𝑐 𝑛 + 𝑏 1 𝑐 𝑛−1 + · · · + 𝑏 𝑛−1 𝑐 1 + 𝑏 𝑛 𝑐 0               (3.151)

所以 𝑝 ∤ 𝑏 𝑛 ，这就迫使 𝑚 = deg(𝑔) = deg( 𝑓 ) = 𝑛，因此 ℎ(𝑥) = 𝑐 ∈ 𝐹 \ {0} 是个非零常数。于是，在 𝐹 [𝑥] 上，
𝑓 (𝑥) 只能被分解为
𝑓 (𝑥) = 𝑐𝑔(𝑥)                                      (3.152)

可是 𝑐 ∈ 𝐹 \ {0} 在 𝐹 上是个单位，在 𝐹 [𝑥] 上也是个单位，所以 𝑓 (𝑥) 在 𝐹 [𝑥] 上一定是不可约的。这就导致
了一个矛盾。因此， 𝑓 (𝑥) 在 𝐹 [𝑥] 上是不可约的。
进一步地，根据上面讲过的引理，当 𝑓 (𝑥) 在 𝑅[𝑥] 上是本原多项式，即含量为 1 时，我们就知道 𝑓 (𝑥) 在 𝑅[𝑥]
上也是不可约的。
我们来看一个具体的例子。
例题 3.1 证明 𝑓 (𝑥) = 𝑥 3 + 20𝑥 2 − 6𝑥 + 2 在 Z[𝑥] 是不可约的。
证明 注意到首项系数是 1，其余系数都是 2 的倍数，而常数项系数不是 4 的倍数。因此，我们可以使用艾森斯
坦判别法。令 𝑅 = Z，而 𝑝 = 2，则所有条件都满足。又因为 𝑓 的含量显然是 1（因为首项系数是 1）
，所以 𝑓 (𝑥)
不仅在 Q[𝑥] 上不可约，更在 Z[𝑥] 上不可约。此即得证。
