# 1.5 群作用

群作用是个较为抽象的概念。为了让大家理解这一概念，请允许我们为大家以更多的概念和例子做些铺垫。
这些铺垫不会是无关信息的堆砌，而是为了在重要的概念登场前，让大家做好准备。
第一个概念是对称群，或置换群。

定义 1.30
令 𝑆 是一个集合，则 𝑆 上的置换群（或对称群），记作 (Perm(𝑆), ◦)，由所有 𝑆 到自身的双射构成，而这
里的运算是映射的复合运算。
Perm(𝑆) = { 𝑓 : 𝑆 → 𝑆 双射}                 (1.100)
♣

证明 首先，映射的复合是满足结合律的。这是根据定义立刻可知的。
单位元是恒等映射，记作 𝑖𝑑，对所有 𝑠 ∈ 𝑆，定义为
𝑖𝑑 (𝑥) = 𝑥                        (1.101)
故显然有，对所有 𝑓 ∈ Perm(𝑆)， 𝑓 ◦ 𝑖𝑑 = 𝑖𝑑 ◦ 𝑓 = 𝑓 。
逆元是根据双射可知的。假如 𝑓 是一个从 𝑆 到自身的双射，则存在其逆映射 𝑓 −1 ，使得 𝑓 ◦ 𝑓 −1 = 𝑓 −1 ◦ 𝑓 = 𝑖𝑑。
综上所述，(Perm(𝑆), ◦) 是个群，称为 𝑆 上的置换群（或对称群）。
还记得我们是怎么在有限群中证明 |𝑥𝐻| = |𝐻| 的吗（𝐻 < 𝐺，𝑥 ∈ 𝐺）
？我们是构造了一个双射 𝜙 𝑥 : 𝐻 → 𝑥𝐻，
定义为，对于 ℎ ∈ 𝐻，
𝜙 𝑥 (ℎ) = 𝑥ℎ                       (1.102)

则其逆为 𝜙 𝑥 −1 : 𝑥𝐻 → 𝐻。
我们将上面的 𝜙 𝑥 延拓到整个群 𝐺，还用 𝜙 𝑥 来表示，即 𝜙 𝑥 : 𝐺 → 𝐺，定义为左乘 𝑥 的运算，即对于 𝑦 ∈ 𝐺，
𝜙 𝑥 (𝑦) = 𝑥𝑦                       (1.103)
，因为我们可以找到它的逆映射，即 𝜙 𝑥 −1 。这是因
这显然是一个双射（但一般来说不是同态，请大家自行验证）
为
(𝜙 𝑥 ◦ 𝜙 𝑥 −1 ) (𝑦) = 𝑥(𝑥 −1 𝑦) = 𝑦          (1.104)
−1
(𝜙 𝑥 −1 ◦ 𝜙 𝑥 ) (𝑦) = 𝑥 (𝑥𝑦) = 𝑦             (1.105)
所以，我们就得到了下面的式子，希望大家可以迅速理解其含义。
(𝜙 𝑥 ) −1 = 𝜙 𝑥 −1                  (1.106)
由于我们对于每一个 𝑥 ∈ 𝐺，都定义了一个双射 𝜙 𝑥 : 𝐺 → 𝐺，即 𝜙 𝑥 ∈ Perm(𝐺)，所以这其实给出了一个映射
𝜙 : 𝐺 → Perm(𝐺)，即对于 𝑥 ∈ 𝐺，我们定义
𝜙(𝑥) = 𝜙 𝑥 ∈ Perm(𝐺)                     (1.107)
要注意，这个映射 𝜙 的定义域和陪域都是群。所以，难道说这是个群同态？答案是肯定的。否则我们就没有讲
的必要了。

命题 1.41
令 (𝐺, ·) 是一个群，我们通过上面的方法定义 𝜙 : (𝐺, ·) → (Perm(𝐺), ◦)，则 𝜙 是个群同态。
♠

证明 证明是很简单的。令 𝑥, 𝑦 ∈ 𝐺，对于 𝑧 ∈ 𝐺，我们有
(𝜙 𝑥 ◦ 𝜙 𝑦 ) (𝑧) = 𝑥(𝑦𝑧) = (𝑥𝑦)𝑧 = 𝜙 𝑥 𝑦 (𝑧)      (1.108)
由于这对于所有 𝑧 ∈ 𝐺 都成立，故
𝜙𝑥 ◦ 𝜙𝑦 = 𝜙𝑥𝑦                        (1.109)





即
𝜙(𝑥𝑦) = 𝜙(𝑥) ◦ 𝜙(𝑦)                                     (1.110)

这就证明了 𝜙 : 𝐺 → Perm(𝐺) 是个群同态。
即使证明很简单，但这样的结论无疑会给每个初学者以“美的体会”
。这便是代数这一门学科美的地方。它
常常给你一些好的结论，而证明不会太难。事实上，它的“美”
，蕴含在对称之中，而群论讲的就是对称的故事。
对于这一种“美”，我们为它起名，称作“群作用”。

定义 1.31
令 (𝐺, ·) 是一个群，𝑆 是一个集合，而 𝜙 : 𝐺 → Perm(𝑆)。若 𝜙 是一个群同态，则我们说 𝜙 是 𝐺 在（集
合)𝑆 上的群作用。
♣

根据上面的命题，我们就知道，“左乘”是一个 𝐺 在其自身的群作用 (也就是 𝑆 = 𝐺 的特例)。
左乘作用似乎过于显然，因此我们给一个稍微不那么显然的例子，也是在自身的作用，叫做“共轭作用”。

定义 1.32
令 (𝐺, ·) 是一个群，我们对 𝑥 ∈ 𝐺，定义 𝜙 𝑥 ∈ Perm(𝐺)，对 𝑦 ∈ 𝐺，定义为
𝜙 𝑥 (𝑦) = 𝑥𝑦𝑥 −1                                  (1.111)
则 𝜙 : 𝐺 → Perm(𝐺)，对 𝑥 ∈ 𝐺，定义为 𝜙(𝑥) = 𝜙 𝑥 ，被称为 𝐺 的共轭作用。
♣

命题 1.42
令 (𝐺, ·) 是一个群，则 𝐺 的共轭作用是 𝐺 在自身的一个群作用。
♠

证明 首先，我们要说明 𝜙 𝑥 是双射，而这是显然的，因为其逆是 𝜙 𝑥 −1 。而这是因为，对于 𝑦 ∈ 𝐺，
(𝜙 𝑥 ◦ 𝜙 𝑥 −1 )(𝑦) = 𝜙 𝑥 (𝑥 −1 𝑦𝑥) = 𝑥(𝑥 −1 𝑦𝑥)𝑥 −1 = 𝑦                (1.112)

(𝜙 𝑥 ◦ 𝜙 𝑦 ) (𝑧) = 𝜙 𝑥 (𝑦𝑧𝑦 −1 ) = 𝑥(𝑦𝑧𝑦 −1 )𝑥 −1 = (𝑥𝑦)𝑧(𝑥𝑦) −1 = 𝜙 𝑥 𝑦 (𝑧)      (1.114)

这就证明了共轭作用确实是一个群在自身的群作用。
事实上，我们必须要注意，共轭作用比左乘作用更“好”
，虽然其看上去更复杂一些。好在什么地方呢？我
们知道，左乘作用的 𝜙 𝑥 ，大部分不是群同态，可是共轭作用的 𝜙 𝑥 ，个个都是同构。因为这一额外的性质，我们
补充一个命题。

命题 1.43
令 (𝐺, ·) 是一个群，𝑥 ∈ 𝐺，则 𝜙 𝑥 : 𝐺 → 𝐺，对 𝑦 ∈ 𝐺，定义为
𝜙 𝑥 (𝑦) = 𝑥𝑦𝑥 −1                                  (1.117)
是一个群 𝐺 的自同构（即到自身的同构）。
♠

证明 我们已经说明，𝜙 𝑥 一定是双射，因为它的逆是 𝜙 𝑥 −1 。因此我们只须证明 𝜙 𝑥 本身还是个同态（不是说 𝜙 是





同态，而是说每个 𝜙 𝑥 是同态）。因此我们令 𝑦, 𝑧 ∈ 𝐺，只须证明 𝜙 𝑥 (𝑦𝑧) = 𝜙 𝑥 (𝑦)𝜙 𝑥 (𝑧)。而这是因为
𝜙 𝑥 (𝑦)𝜙 𝑥 (𝑧) = (𝑥𝑦𝑥 −1 ) (𝑥𝑧𝑥 −1 ) = 𝑥(𝑦𝑧)𝑥 −1 = 𝜙 𝑥 (𝑦𝑧)      (1.118)
恰好约掉。这就证明了共轭作用下的每一个 𝜙 𝑥 都是群 𝐺 的自同构。
，意指由群 𝐺 内的
注意，我们对这样的自同构很感兴趣。因此再给它们一个特殊的名字，叫做“内自同构”
元素引出的自同构。而所有其他的自同构，就叫做“外自同构”。

定义 1.33
令 (𝐺, ·) 是一个群，则一个 𝐺 的（由 𝑥 ∈ 𝐺 引出的）内自同构，指的是 𝜙 𝑥 : 𝐺 → 𝐺，对 𝑦 ∈ 𝐺，定义为
𝜙 𝑥 (𝑦) = 𝑥𝑦𝑥 −1                         (1.119)
而其他所有 𝐺 上的自同构，则称为 𝐺 上的外自同构。
♣

我们回到群作用。囿于篇幅，我们希望再讲述一个定理，称为“轨道-稳定化子定理”。
注意，一般来说，令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用，𝑥 ∈ 𝐺，𝑠 ∈ 𝑆，则我们会用 𝑥 · 𝑠，甚
至 𝑥𝑠，来代表 𝜙 𝑥 (𝑠)，或 𝜙(𝑥, 𝑠)。在这样的记号下，群作用的性质等价地变成下面两条。

命题 1.44
令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用，假如我们用 𝑥 · 𝑠，甚至 𝑥𝑠，来代表 𝜙 𝑥 (𝑠)，或 𝜙(𝑥, 𝑠)
（其中 𝑥 ∈ 𝐺，𝑠 ∈ 𝑆），则我们等价地，可以把群作用的性质，记作
∀𝑠 ∈ 𝑆, 𝑒 · 𝑠 = 𝑠                 (1.120)
∀𝑥, 𝑦 ∈ 𝐺, 𝑥 · (𝑦 · 𝑠) = (𝑥𝑦) · 𝑠                 (1.121)
或者进一步地，在不会引起歧义的情况下，记作
∀𝑠 ∈ 𝑆, 𝑒𝑠 = 𝑠                     (1.122)
∀𝑥, 𝑦 ∈ 𝐺, 𝑥(𝑦𝑠) = (𝑥𝑦)𝑠                       (1.123)
♠

证明 若 𝜙 是一个群作用，则显然利用同态的性质我们有第二条。而根据同态把单位元映到单位元，我们有 𝜙𝑒 = 𝑖𝑑，
即对所有 𝑠 ∈ 𝑆，𝑒𝑠 = 𝑠。这就证明了一个方向。
另一方向，若 · : 𝐺 × 𝑆 → 𝑆 满足这两条性质，我们想要定义 𝜙 : 𝐺 → Perm(𝑆)。当然对于 𝑥 ∈ 𝐺, 𝑠 ∈ 𝑆，定义
𝜙(𝑥)(𝑠) = 𝜙 𝑥 (𝑠) = 𝑥𝑠                          (1.124)
我们当然也证明这样的 𝜙 𝑥 是双射，否则都不是良定义的。但这几乎是显然的。
𝑥(𝑥 −1 𝑠) = (𝑥𝑥 −1 )𝑠 = 𝑒𝑠 = 𝑠                      (1.125)
𝑥 −1 (𝑥𝑠) = (𝑥 −1 𝑥)𝑠 = 𝑒𝑠 = 𝑠                      (1.126)
注意，上面两行的最后一步我们用到了命题中的第一条性质，为了证明 𝜙 𝑥 是双射，这一条性质是不可或缺的。

因此命题中的第一条性质，是说明 𝜙 是良定义的（𝜙 𝑥 是双射），而第二条性质是说明 𝜙 是同态。二者缺一
不可。这两条性质加起来，就是群作用的定义。
接下来，我们举一个例子，地球的自转。我们知道地球是沿着一根轴自转的，我们固定这个轴。那么每一种
自转对应着一个角度，其自由度为 1。我们把所有的自转构成的群叫做 𝐺。而每一个自转，都会把地球表面的一
个地方，带到另一个地方。
什么是轨道呢？我们取定地球上一个地点。那么每一个自转都会作用在这个地点，把它带到另一个地点。所
有能够在群作用下“去到的地方”
，就构成了轨道。我们不难发现，地球表面每个地方的轨道，就是纬度相同的




一圈。
稳定化子是什么呢？我们同样取定地球上一个地点。有没有哪个自转作用在这个地点后位置还不变的？有
的。那些转了 2𝜋𝑛(𝑛 ∈ Z) 弧度的自转，是那些“固定”了这个地点的自转，也可以说“稳定”了这个地点。
假如记得没错的话，这是我们第一次引入借助生活中的例子来理解抽象代数。希望大家可以认识到，群作
用是一个普遍的现象。下面我们给出严格定义。

定义 1.34
令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用。若 𝑠 ∈ 𝑆。则我们定义 𝑠 的轨道，记作 Orb(𝑠)，定义
为
Orb(𝑠) = {𝑠 ′ ∈ 𝑆 : ∃𝑥 ∈ 𝐺, 𝑠 ′ = 𝑥𝑠} = {𝑥𝑠 : 𝑥 ∈ 𝐺}                 (1.128)

Stab(𝑠) = {𝑥 ∈ 𝐺 : 𝑥𝑠 = 𝑠}                             (1.129)
♣

我们借地球自转的例子，可以发现轨道与轨道之间应当是要么相等，要么无交的。我们来证明这一命题。

命题 1.46
令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用，而 𝑠 ∈ 𝑆，则 𝑠 的稳定化子是 𝐺 的子群，即

♠

证明 一，𝑒𝑠 = 𝑠。二，若 𝑥, 𝑦 ∈ Stab(𝑠)，则 (𝑥𝑦)𝑠 = 𝑥(𝑦𝑠) = 𝑥𝑠 = 𝑠。三，若 𝑥𝑠 = 𝑠，则左乘 𝑥 −1 ，得到 𝑥 −1 𝑠 = 𝑠。



注意稳定化子一般不是正规子群。这里我们省略反例。

引理 1.5
令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用，𝑠 ∈ 𝑆，𝑥, 𝑦 ∈ 𝐺，则 𝑥𝑠 = 𝑦𝑠 当且仅当 𝑥 −1 𝑦 ∈ Stab(𝑠)。
♡

证明 对 𝑥𝑠 = 𝑦𝑠 两边同时左乘 𝑥 −1 ，就显然了。
这个引理的作用即在于下面的轨道-稳定化子定理

命题 1.47 (轨道-稳定化子定理)
令 𝜙 : (𝐺, ·) → (Perm(𝑆), ◦) 是一个 𝐺 在 𝑆 的群作用，𝑠 ∈ 𝑆，则存在 𝐺/Stab(𝑠) 到 Orb(𝑠) 的双射。
特别地，若 𝐺 是有限群，则
|𝐺 | = | Stab(𝑠)| · | Orb(𝑠)|                  (1.132)
♠

证明 令 𝑓 : 𝐺/Stab(𝑠) → Orb(𝑠)，定义为 𝑓 (𝑥 Stab(𝑠)) = 𝑥𝑠。
首先证明 𝑓 是良定义的。根据上面的引理，若 𝑥 Stab(𝑠) = 𝑦 Stab(𝑠)，则 𝑥 −1 𝑦 ∈ Stab(𝑠)，故 𝑥𝑠 = 𝑦𝑠。
根据 Orb(𝑠) 的定义， 𝑓 显然是一个满射。
单射则是再次利用上面的引理。若 𝑥𝑠 = 𝑦𝑠，则 𝑥 −1 𝑦 ∈ Stab(𝑠)，故 𝑥 Stab(𝑠) = 𝑦 Stab(𝑠)。
假如 𝐺 是有限群，则同时取集合大小，就得到了
|𝐺 | = | Stab(𝑠)| · | Orb(𝑠)|                     (1.133)
综上，我们就证明了轨道-稳定化子定理。
有时，我们不知道一个有限群 𝐺 的大小，但知道它作用在一个集合 𝑆 上，那我们就可以利用轨道-稳定化子
定理，找到某一个点 𝑠 ∈ 𝑆 的轨道和稳定化子，进而算出 𝐺 的阶。除了算出阶，其实也可以通过完全知道一个群
的元素有哪些。例如 𝐺 有 𝑛 个元素，而你恰好找到了 𝑛 个不相等的元素，那么这个群 𝐺 一定就是由你找到的那
些元素来构成的，无一例外，无一多余。
我们来看一个例子，二面体群 𝐷 2𝑛 ，它是由所有正 𝑛 边形到自身的对称变换所构成的。什么是对称变换？就
是把自身映到自身，而且是保距的。保距指的是，原先距离相同的点，变换后距离仍然相同。如图1.7中的例子
所示。

事实上，每一个对称变换由其 𝑛 个顶点的像唯一确定，因为其余的点都可以通过顶点来找到位置。很明显，
在初中的时候我们就知道，这是个轴对称图形，有 𝑛 个翻折变换；这还是个中心对称图形，有 𝑛 个旋转变换。那

命题 1.48
|𝐷 2𝑛 | = 2𝑛
♠

证明 任取正多边形的一个顶点 𝑠，考虑其轨道 Orb(𝑠)。最多只有 𝑛 个顶点可以去，而 𝑛 个旋转变换恰好带 𝑠 去
了这些顶点，因此 | Orb(𝑠)| = 𝑛。





接下来，考虑其稳定化子 Stab(𝑠)。如果 𝑥 ∈ 𝐷 2𝑛 把 𝑠 映射到 𝑠，但又有保证是一个等距变换，则 𝑠 相邻的两

出，这两个是唯二的 𝑠 的稳定化子。因此 | Stab(𝑠)| = 2。
根据轨道-稳定化子定理，|𝐷 2𝑛 | = | Orb(𝑠)| · | Stab(𝑠)| = 2𝑛。这就证明了这个命题。
因此，二面体群 𝐷 2𝑛 就是恰好由 𝑛 个翻折变换和 𝑛 个旋转变换所组成的群。
