## 循环求余
+ 根据求余运算性质推出（∵ 本题中 ![image](https://g.yuque.com/gr/latex?x%3Cp)，∴ ![image](https://g.yuque.com/gr/latex?x%20%5Codot%20p%20%3D%20x) ）：

**解析**： 利用此公式，可通过循环操作依次求 ![image](https://g.yuque.com/gr/latex?x%5E1%2C%20x%5E2%2C%20...%2C%20x%5E%7Ba-1%7D%2C%20x%5Ea)对 p 的余数，保证每轮中间值 rem 都在 int32 取值范围中。封装方法代码如下所示。

```python
# 求 (x^a) % p —— 循环求余法
def remainder(x, a, p):
    rem = 1
    for _ in range(a):
        rem = (rem * x) % p
    return rem
```

+ 根据求余运算性质可推出：

+ 当 a为奇数时 a/2不是整数，因此分为以下两种情况（ ''`//`' 代表向下取整的除法）：



![image](https://cdn.nlark.com/yuque/__latex/b91ed89680af839920cb444d1c28c631.svg)



**解析**： 利用以上公式，可通过循环操作每次把指数 a 问题降低至指数 `a//2 `问题，只需循环 ![image](https://g.yuque.com/gr/latex?log_2(N))次，因此可将复杂度降低至对数级别。封装方法代码如下所示。

```python
# 求 (x^a) % p —— 快速幂求余
def remainder(x, a, p):
    rem = 1
    while a > 0:
        if a % 2: rem = (rem * x) % p
        x = x ** 2 % p
        a //= 2
    return rem
```

**帮助理解：**这里`rem` 记录者每次`a`为奇数的时候多的那一项`x` ，最后会将 ![image](https://g.yuque.com/gr/latex?rem%20%5Ctimes%20(x%5Ea%20%5Codot%20p)) 化为 ![image](https://g.yuque.com/gr/latex?rem%20%5Ctimes%20(x%5E0%20%5Codot%20p)%20%3D%20rem%20%5Ctimes%201)的形式，即 `rem` 为余数答案。
